| boring 与 bored |          |     |
| -------------- | -------- | --- |
| 词条             | 相同点      | 不同点 |
| bored          | 形容词      | 修饰人 |
| boring         | 由bored变来 | 修饰物 |

| 原型     | 比较级   | 最高级    | 用途       | 译义   |
| ------ | ----- | ------ | -------- | ---- |
| many   | more  | most   | + 可数名词   | 多    |
| much   | more  | most   | + 不可数名词  | 多    |
| little | less  | least  | + 不可数名词  | 几乎没有 |
| few    | fewer | fewest | + 可数名词复数 | 几乎没有 |

| ------------ | --------- |

| die          | be dead   |
| leave        | be away   |
| become/join  | be in     |
| borrow       | keep      |
| buy          | have      |
