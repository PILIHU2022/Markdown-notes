# 运算符

比较与Python相同。

## 分配

适用于变量
| Operator | Example   | Same As      |
|:--------:|:---------:|:------------:|
| `=`      | `x = 5`   | `x = 5`      |
| `+=`     | `x += 3`  | `x = x + 3`  |
| `-=`     | `x -= 3`  | `x = x - 3`  |
| `*=`     | `x *= 3`  | `x = x * 3`  |
| `/=`     | `x /= 3`  | `x = x / 3`  |
| `%=`     | `x %= 3`  | `x = x % 3`  |
| `&=`     | `x &= 3`  | `x = x & 3`  |
| `|=`     | `x |= 3`  | `x = x | 3`  |
| `^=`     | `x ^= 3`  | `x = x ^ 3`  |
| `>>=`    | `x >>= 3` | `x = x >> 3` |
| `<<=`    | `x <<= 3` | `x = x << 3` |

## 逻辑运算符
| C    | Python   |
|:----:|:--------:|
| `&&` | `and`    |
| `!`  | `not`    |
| `||` | `or `    |
